# X-Ray_Xerxes1138

An attempt at converting X-Ray 1.0 to PBR.

# FEATURES
* SSAO (Crytek)
* Stochastic Screen Space Reflection
* Screen SubSurface Scattering
* Temporal AA
* Motion Vector (Camera Only)
* Contact Shadows
* Tonemap (ACES or Hable)
